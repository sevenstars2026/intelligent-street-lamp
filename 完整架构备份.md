## 一、完整功能清单

### 1.1 设备控制模块
- **单设备控制**：POST /api/devices/:deviceId/control
- **批量控制**：POST /api/devices/batch-control
- **控制日志**：GET /api/devices/:deviceId/control-logs
- **控制超时机制**：5秒超时自动标记失败

### 1.2 自动化控制模块
- **模式管理**：
  - GET /api/devices/:deviceId/mode
  - PUT /api/devices/:deviceId/mode (auto/manual)
- **阈值配置**：
  - GET /api/devices/:deviceId/threshold
  - POST /api/devices/:deviceId/threshold
- **自动化算法**：5样本4触发阈值算法
  - 收集最近5次光照数据
  - 当4次以上满足条件时触发
  - 5分钟冷却期防止频繁切换

### 1.3 告警管理模块
- **告警接口**：
  - GET /api/alarms (列表查询，支持多维度筛选)
  - GET /api/alarms/:alarmId (详情)
  - PUT /api/alarms/:alarmId/resolve (处理告警)
- **告警类型**：
  - device_offline：设备离线
  - control_failed：控制失败
  - sensor_abnormal：传感器异常
- **告警级别**：
  - low：低级
  - medium：中级
  - high：高级
  - critical：紧急
- **告警升级算法**：
  - 24小时未处理：low → medium
  - 48小时未处理：medium → high
  - 72小时未处理：high → critical

### 1.4 数据统计模块
- **设备统计**：GET /api/devices/:deviceId/statistics
  - 控制成功率
  - 平均光照强度
  - 在线率
  - 告警统计
- **全局统计**：GET /api/statistics/overview
  - 设备总数/在线数/离线数
  - 告警总数/活跃告警数
  - 平均控制成功率
  - 所有设备详细统计列表
- **数据聚合算法**：
  - 按小时聚合：计算avg/max/min/count
  - 按天聚合：基于小时数据二次聚合
  - 支持窗口滑动计算

### 1.5 光照数据模块
- **历史查询**：GET /api/devices/:deviceId/light-history
  - 时间范围筛选
  - 聚合方式：raw/hourly/daily
  - 分页支持

### 3.2 告警升级算法

```typescript
/**
 * 告警升级规则
 * - 24h: low → medium
 * - 48h: medium → high
 * - 72h: high → critical
 */
upgradeAlarms() {
  const now = new Date()
  alarms.filter(a => a.status === 'active').forEach(alarm => {
    const hours = (now.getTime() - alarm.createdAt.getTime()) / (1000 * 3600)
    
    if (hours >= 72 && alarm.alarmLevel !== 'critical') {
      alarm.alarmLevel = 'critical'
    } else if (hours >= 48 && alarm.alarmLevel === 'low') {
      alarm.alarmLevel = 'high'
    } else if (hours >= 24 && alarm.alarmLevel === 'low') {
      alarm.alarmLevel = 'medium'
    }
  })
}
```

/**
 * 时间窗口聚合
 * - 按小时/天聚合原始数据
 * - 计算avg/max/min/count
 */
aggregateByHour(deviceId: string, startTime: Date, endTime: Date) {
  const rawData = getLightDataInRange(deviceId, startTime, endTime)
  
  // 按小时分组
  const grouped = groupByHour(rawData)
  
  // 计算统计值
  return grouped.map(group => ({
    avgLightIntensity: average(group.values),
    maxLightIntensity: max(group.values),
    minLightIntensity: min(group.values),
    dataCount: group.values.length,
    timeWindow: group.hour
  }))
}
